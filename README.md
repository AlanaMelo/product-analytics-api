# Product Analytics API

Production-ready backend API designed to simulate how companies track user behavior and product performance.

## Features
- User management
- Event tracking system
- Product analytics metrics (DAU)
- Scalable architecture

## Tech Stack
- Python
- FastAPI
- SQLAlchemy

## Example Use Case
This API simulates how digital products track user behavior and generate insights like engagement and activity.
